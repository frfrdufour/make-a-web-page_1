Structure of Webpages

Pages are made of rectangle boxes. Each one can have its own style. It's best to give them a class name as attribute.

Lesson 3
How HTML works with CSS

How to include CSS Styling in my web page?

Method 1: write CSS in the "head" of your HTML

This method is good for very small projects (like what you'd do in scratchpad.io). To do this: 1. At the top of your HTML document, add a "head" element. 2. Inside the "head" element, add a "style" element. 3. Write your CSS in the "style" element and then put the HTML for the rest of your page below.

Method 2: Link your HTML to a separate CSS file

This adds another step, but it lets you stay more organized when working on larger projects. To do this:

  1. Write all of your structural HTML in one file (let's call itmain.html).
  2. In a separate file (let's call it main.css), put all of your CSS code.
  3. Add a "head" element to the top of your HTML.
  4. Add a "link" tag inside the "head" element. Since "link" is a "void tag" you don't need to add a closing "/link".
  5. Add the following attributes to your "link": * rel="stylesheet" *type="text/css" * href="main.css"

Method 3: Write your style inline with your HTML

This is generally a very bad idea because it leads to lots of repeated code

The Box Model

Margin, Border, padding & content

The padding clears an area around the content. It’s affected by the background color of the box.
The border is inherited by the color property of the box.
The margin is the outer layer and clears an area around the border. Does not have a background color and is completely transparent, like a space btw boxes.
So the dimension of your element = content + padding + border. I.e. we ignore margin.

Box Sizing

There are two techniques you can use to help deal with sizing issues:

  1. Set sizes in terms of percents rather than pixels.
  2. Set the box-sizing attribute to border-box for every element.

Box Positioning

  1. Divs are block elements (as opposed to inline), so by default they take up the entire width of a page.
  2. Adding the rule display: flex; to the appropriate CSS will override this behavior and let divs appear next to each other.
